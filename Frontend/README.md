# AI Resume Maker - Frontend

Next.js React frontend for the AI Resume Maker application.

## Setup

```bash
cd Frontend
npm install
npm run dev
```

The frontend runs on `http://localhost:3000`

## Features

- User authentication (login/register)
- Resume builder with AI-powered content generation
- Dashboard to manage resumes
- Beautiful UI with Tailwind CSS

## Configuration

Set your backend API URL in `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:5000
```
